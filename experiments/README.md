# Experiments Directory

This directory contains experimental code, comparisons, tests, and demos developed during the project.

## Structure

- `comparisons/` - Library comparison scripts and benchmarks
- `tests/` - Test scripts for optimization implementations
- `demos/` - Demo scripts and setup utilities

- Comprehensive library performance analysis
- Enhanced comparison scripts with detailed metrics

### Tests
- Practical optimization tests

- Performance testing utilities

### Demos
- Optimization demonstrations
